Unit Circle Ferris Wheel Project Junoir Project

Over the course of a few class periods, my math teacher assigned us into groups and to build a ferris wheel out of Kenix pieces. Later, we learned that all the degrees on the ferris wheel match up to the units on a unit circle. This project was very fun and engaged me more into the topic. Relating things we learn to the real world, I think, has a better understanding to it then not knowing why or how exactly the material is used outside the classroom.

Geometry iBook Project Freshman Project

This is a project we had to do in my honors math class, geometry. The point of this project was to pick a topic in geometry of your choice and write a children's book about it. In the book you must teach the grade level of your choice about the subject you have chosen. I picked this project as my artifact to add in my digital portfolio because I very much enjoyed creating this project and I put a lot of hard work into it.
